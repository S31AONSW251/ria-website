# RIA Website Images

Upload website images here. Files inside `public` can be used directly by the React app.

Recommended structure:

- `public/images/logos/ria-conscious-logo.png`
- `public/images/hero/ria-hero.jpg`
- `public/images/cards/product.jpg`
- `public/images/cards/research.jpg`
- `public/images/cards/safety.jpg`
- `public/images/cards/memory.jpg`
- `public/images/cards/emotion.jpg`
- `public/images/cards/journal.jpg`

Use `.jpg`, `.png`, or `.webp`.

After uploading, refresh the site. If the image filename matches the code path, it will load automatically.
